## Features

## Installation

### From VS Code Marketplace
1. Open VS Code
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Type `ext install digitalforge.f26f0664`
4. Press Enter

### From VSIX file
1. Download the `.vsix` file
2. In VS Code: `Extensions > ··· > Install from VSIX`
3. Select the downloaded file

## Usage

Press `Ctrl+Shift+P` and search for `Cursor Rules & AI IDE Linting and Best Practices Enforcer VS Code Extension` to see all available commands.

**Keyboard shortcut:** `Ctrl+Shift+Alt+A` (or `Cmd+Shift+Alt+A` on Mac)

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `f26f0664.enabled` | `true` | Enable/disable the extension |
| `f26f0664.autoRun` | `false` | Auto-run on file open |
| `f26f0664.logLevel` | `"info"` | Log verbosity |
